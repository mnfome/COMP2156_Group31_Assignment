SDLC (Software Development Life Cycle) is a structured process used to design, develop, test, and deploy software efficiently and with high quality.
Phases of SDLC
1️⃣ Planning – Define project scope & goals.
2️⃣ Requirement Analysis – Gather and document user needs.
3️⃣ Design – Create system architecture & UI design.
4️⃣ Implementation (Coding) – Developers write the code.
5️⃣ Testing – Find and fix bugs before release.
6️⃣ Deployment – Release software for users.
7️⃣ Maintenance – Continuous updates & support.
